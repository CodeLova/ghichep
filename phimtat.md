Các phím tắt hay dùng
====================

# Trên Terminal của Ubuntu
Phím tắt gọi terminal
> <kbd>CTL</kbd> + <kbd>ALT</kbd> + <kbd>t</kbd>

Phím tắt copy
> <kbd>CTL</kbd> + <kbd>ALT</kbd> + <kbd>C</kbd>

 Phím tắt paste

<kbd>CTL</kbd> + <kbd>ALT</kbd> + <kbd>V</kbd>
